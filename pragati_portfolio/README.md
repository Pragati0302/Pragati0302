This is a Vite + React + TailwindCSS portfolio website for Pragati Priyadarshini.

## Quick start (locally)

1. Install dependencies:
```bash
npm install
```

2. Run dev server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Deploy

- **Netlify / Vercel**: Connect this repository and set the build command to `npm run build` and publish directory to `dist`.
- **GitHub Pages**: Build and use a static host pointing to `dist/`.
